### VPC Endpoint란
VPC 엔드포인트는 VPC 내 Resource들이 VPC 외부의 서비스 등에 접근할 때 Internet Gateway, NAT Gateway 등의 외부 인터넷 전송 서비스를 타지 않고 내부 네트워크를 통해 접근할 수 있도록 지원하는 서비스
